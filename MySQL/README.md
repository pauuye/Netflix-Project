**Netflix Customer Analysis**



This project is a simple Exploratory Data Analysis (EDA) using SQL.

I analyzed a Netflix customer dataset to understand user behavior, churn rate, and viewing patterns.



**Tools Used**



**MySQL**



**Basic SQL functions: GROUP BY, CASE WHEN, CTE, WINDOW FUNCTIONS**





**What I Did**



Checked duplicates and data quality



Created Age Brackets



Analyzed churn patterns



Checked watching behavior



Identified most-used devices



Compared payment methods



Looked at regional performance



Computed revenue per region
