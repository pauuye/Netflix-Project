**Netflix Customer Churn Dashboard**



**Overview**



This folder contains the Power BI report developed to visualize and analyze customer churn patterns for a Netflix-style subscription platform. The report consolidates key churn drivers into an interactive dashboard to support data-driven insights.



**Purpose**



This Power BI report demonstrates interactive data visualization, DAX-based metrics, and analytical storytelling to support churn analysis and retention-focused insights.





**File Description**



File: Churned Analysis.pbix



Description: Power BI report containing data modeling, DAX measures, and visualizations focused on customer churn behavior.



**Data Model**



The dataset is imported from the cleaned churn CSV file.



A date table is used to support time-based analysis.



Relationships are established to enable accurate filtering across customer attributes such as age group, subscription plan, device type, and genre.



Dashboard Highlights



Overall churn rate KPI



Churn distribution by age group



Churn comparison by subscription plan



Device usage patterns and associated churn rates



Genre preference analysis with churn indicators



**Key Insights**



Identification of age group with high-risk of churning.



Visibility into subscription plans with elevated churn rate.



Understanding of device usage and content preference impact on churn rate.



Understanding of genre watched impact on churn rate.
